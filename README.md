
# Проверка сложности пароля
Это простая программа на Python, которая проверяет, насколько надежный пароль придумал пользователь. 
## Что программа умеет:
* **Проверка по пунктам:** Проверяет длину, цифры, большие и маленькие буквы, а также знаки.
* **Удобный ввод:** Программа работает в цикле, так что можно проверять много паролей подряд (не надо каждый раз запускать заново).
* **Обработка ошибок:** Если просто нажать Enter (пустой ввод), программа не вылетит, а попросит ввести пароль нормально.
* **Красивый вывод:** Используются эмодзи, чтобы сразу было видно, где ошибка, а где всё ок.
## Как запустить:
1. Установите Python.
2. Скачайте файл `main.py`.
3. Откройте терминал или консоль в папке с файлом.
4. Напишите команду:
   ```bash
   python main.py
   ```
## ПРИМЕР РАБОТЫ
```text
Введите пароль (или 'exit' для выхода):
>> qwerty123

Результаты:
  ✅ Длина от 8 символов
  ✅ Наличие цифр
  ❌ Заглавные буквы (A-Z)
  ✅ Строчные буквы (a-z)
  ❌ Спецсимволы (@#$%^&*)
------------------------------
ИТОГ: Средний пароль. Можно лучше.
```
## есть:
* Библиотека `re` для поиска символов.
* Цикл `while`, чтобы программа не закрывалась.
* Блок `try-except` для защиты от ошибок.
* Разделение кода на функции, чтобы всё было аккуратно.
